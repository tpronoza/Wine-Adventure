## Use<!-- -->
Wine Adventure is an app for people that love their wine.

## Features<!--  -->

###### User is able to add, edit and delete wines that were added by them
###### User is able to Search for a specific Wine that they are looking for
###### User is able to add, edit and delete wines 101 articles that were added by them
###### Wines can be added and delete to/from Favorite list
###### Wines can be added and deleted to/from Wishlist try later
###### Wines can be added and to/from WineList (wine that user tried before)

#### Features in Progress
###### - User will be able to rate wines and leave a comment
###### - User will be able to create and view their profile
###### - Will be able to filter by Wine by categories (Ex. Red, White, Rose, Sparkling, Dessert and etc.)
###### - Sort Results by Ex. User Rating or 
###### - Should be able to view other user profiles
###### - Should be able to follow other users
